# React Gradient Border Card Component

A sleek, reusable React card component styled entirely with Tailwind CSS. It features a unique gradient border, embedded image support, and smooth hover animations.

## Features
- **Gradient Border:** Uses a wrapper technique to achieve a beautiful gradient border while perfectly maintaining rounded corners.
- **Image Integration:** Displays an image beautifully nestled between the header and description.
- **Interactive:** Built-in micro-interactions, including a subtle card lift on hover and an image zoom effect.
- **Single File:** Easy to copy and paste directly into your project without juggling multiple CSS files.
- **Fully Responsive:** Tailwind utility classes ensure it adapts smoothly to different screen layouts.

##  Installation & Usage

1. Create a new file in your project, for example, `CardComponent.jsx`.
2. Copy the provided code (including the `Card` and `App` components) into this file.
3. Import and render the component wherever needed in your application.

##  Customization Guide

Tailwind makes it incredibly easy to tweak this component to fit your specific theme:

**Change the Gradient Colors:** Locate `bg-gradient-to-br from-pink-500 to-purple-600 and green-400 on the outer `div` and replace the `from-*` and `to-*` classes with your preferred brand colors.
**Adjust Border Thickness:** The border thickness is controlled by the padding on the outer div (`p-1`, which equals 4px). Change it to `p-0.5` (2px) for a thinner border,  (8px) for a thicker one.
**Modify Card Width:** Update the `w-80` class on the parent `div` to adjust the overall size of the card.
